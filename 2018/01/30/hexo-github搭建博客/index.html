hexo+github搭建博客

发表于 2018-01-30 | 分类于 学习笔记

概述

我在网上看了不少博客，但是没有一个能让我完整搭建起来一个博客的，还是多亏了大佬明的帮助，我才把博客搭建起来。下面写一下我搭建博客的步骤。以下步骤是在默认你安装git下进行的。

搭建步骤

1. 安装node js

hexo基于node js，因此要先安装node js。先进入官网，链接如下：https://nodejs.org/en/download/ ，下载源码包，即Source Code下的node-v8.9.4.tar.gz文件。
将该文件放在~/下(好像说是放在中文文件夹下会安装失败），然后依次运行如下命令

tar zxvf node-v8.9.4.tar.gz    //解压缩
 cd node-v8.9.4/    //进入该文件夹
 ./configure        //执行该文件夹下的可执行文件
 make -j4    //编译
 sudo make install    //安装编译好的源码包

至此node js就安装完毕了

2. 安装hexo

运行如下命令

npm install hexo-cli -g    
 hexo init blog    //建立一个名为blog的文件夹并将其初始化
 cd blog
 npm install    
 hexo s    //运行博客

当执行完最后一条指令时，如果显示如下语句，

INFO  Hexo is running at http://localhost:4000/. Press Ctrl+C to stop.

说明你成功了，这时候你访问http://localhost:4000/ 网址即可看到你的博客。注意：hexo命令以及部分npm命令一定要在blog目录下运行（即你安装hexo的那个目录)。

3. 连接到github

  • 创建github仓库并配置git

    仓库名一定要是你的注册名.github.io，建好仓库之后配置git，输入以下命令
git config --global user.name "github注册名"
 git config --global user.eamil "github注册邮箱"

现在git就配置好了

  • 配SSH

    先生成SSH密钥，输入以下命令
ssh-keygen -t rsa -C "github注册邮箱"

复制~/.ssh/idrsa.pub里的内容，进入github，点击setting->SSH keys添加一个新的SSH密钥，将刚才复制的内容粘贴到里面就可以了。

  • 修改配置文件

    打开~/blog/_config.yml文件，修改其中的内容，下面是我的修改内容
url: AFoolFlypig.github.io    //将这里改为你的仓库名
 root: /
 permalink: :year/:month/:day/:title/        

type: git
   repository: https://github.com/AFoolFlypig/AFoolFlypig.github.io.git
   branch: master

上面这个配置每次部署的时候都要输github帐号密码，下面这个配置就厉害了，不用输帐号密码，直接就能部署。

deploy:
 type: git
 #仓库
 repo: 
   github: git@github.com:AFoolFlypig/AFoolFlypig.github.io.git
 branch: master

然后进入blog目录，执行以下命令

npm install hexo-deployer-git --save 
 hexo g    
 hexo d
 hexo s

就能访问你的博客了。但是现在只是一个粗坯，还需要进一步的配置。至于如何配置，网上有很多教程，我这里就不说了
